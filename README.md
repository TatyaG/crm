<h1 align="center">CRM-система</h1>

<h2>Для работы приложения необходимо запустить сервер</h2>

Перед запуском убедитесь, что вы установили Node.js версии 12 или выше.

Для запуска сервера перейдите в папку с репозиторием и выполните команду `node index`. Для остановки нажмите сочетание клавиш CTRL+C.

<h2>Для запуска приложения необходимо установить веб-сервер <a href="https://www.npmjs.com/package/serve">serve</a></h2>

Чтобы установить его, необходимо выполнить команду:

`npm install --global serve`

Для запуска сервера выполните команду:

`serve`

<h3>Добавления нового клиента</h3>

<p>Для добавления нового клиента необходимо ввести данные в обязательные поля: Фамилия и Имя.</p>
<p></p>Необязательное поле: Отчество.</p>
<p></p>Также есть возможность добавления максимум 9 контактов клиента. Тип контакта можно выбрать из выпадающего списка.</p>

<img width="100%" align="center" src="https://github.com/TatyaG/crm/blob/main/readme-assets/new.png">

<h3>Изменение данных клиента</h3>

<p>При нажатии на кнопку Изменить открывается окно с данными клиента и его контактами, если они присутствуют.</p>
<p>После изменения данных клиента, нажмите на кнопку Сохранить.</p>
<p>Чтобы удалить клиента нажмите кнопку Удалить.</p>

<img width="100%" align="center" src="https://github.com/TatyaG/crm/blob/main/readme-assets/change.png">

<h3>Поиск клиента</h3>

<p>Чтобы выполнить поиск клиента, введите в поле поиска данные клиента (фамилию, имя или отчество).</p>
